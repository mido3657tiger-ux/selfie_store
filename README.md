# 📱 سيلفي ستور | Selfie Store - المنصورة

> **الموقع الإلكتروني الرسمي لبراند "سيلفي ستور" الفخم بقلب شارع العباسي بالمنصورة المتخصص في تجارة الهواتف الذكية الأصلية وأجهزة كسر الزيرو فئة أولى.**

---

## 🛠️ المواصفات التقنية للمشروع
* **واجهة المستخدم (UI):** مبنية بالكامل باستخدام **Tailwind CSS** بنظام الـ Dark Mode الفاخر المعتمد لدى الشركات التكنولوجية الكبرى.
* **محرك الداتا والسيستم الذكي (Core JS):** ملف `app.js` مخصص لإدارة المنتجات ديناميكياً مع توليد رسايل حجز تلقائية ومشفرة ومباشرة لـ **أبو عبد الله** على الواتساب والمكالمات.
* **الأداء والسرعة:** متوافق 100% مع معايير الـ SEO وظهور السيرفرات السريع لضمان أعلى Rank في محركات البحث وجوجل في المنصورة والدقهلية.

---

## 📂 هيكل ملفات المشروع على GitHub
1. `index.html` -> الهيكل البنائي الكامل والأقسام الرئسية للموقع.
2. `style.css` -> اللمسات الجمالية، حركة التصفح الناعمة، وتخصيص الـ Scrollbar الفخم.
3. `app.js` -> مخزن المنتجات، تحديد الأسعار، الأجهزة المتاحة بالمحل وسيستم الحجز السريع.
